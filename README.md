# Clone Spotify Project
This is my Spotify clone project built using MERN stack.

📌 Overview

This project is a Spotify-inspired music streaming web application built using the MERN stack (MongoDB, Express.js, React.js, Node.js).

It allows users to stream music, create playlists, and enjoy a modern music player experience.
Music playback is handled using real-time streaming via YouTube embed integration.

🚀 Features Implemented
🔐 Authentication
User Signup & Login (JWT-based)
Secure authentication system
Persistent sessions


🎵 Music Streaming

🏠 Homepage
